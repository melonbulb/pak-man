# pak-man

[Play]("https://melonbulb.github.io/pak-man/src/index.html")

Game inspired by the classic Pac-Man arcade experience.

# features

1. Player Movement – Smooth navigation in all directions through maze-like paths.
2. Maze Design – Unique layout and pathway.
3. Collectibles – Eat pellets and power-ups scattered across the map.
4. Enemy AI – Ghosts patrolling the map
5. Power Mechanism – Temporary speed boost
6. Score System – Points for eating pellets and power-ups

# How to play

1. Use the arrow keys to move
2. Collect items on tile to increase score
3. Collect power items for speed boost
4. To win, all item must be collected
5. Avoid the ghost at all costs
